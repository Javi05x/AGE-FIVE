# Carpeta `data/`

Esta carpeta contiene los archivos Excel con los datos de los jugadores descargados de Kaggle.
**Esta carpeta está incluida en `.gitignore`** por el tamaño de los archivos y no se sube al repositorio.

## Archivos necesarios

Descarga los siguientes datasets y colócalos en esta carpeta con los nombres indicados:

| Archivo | Temporada | Enlace |
|---|---|---|
| `top5-players-2324.xlsx` | 2023/24 | [Kaggle — Top 5 Leagues 23/24](https://www.kaggle.com/datasets/orkunaktas/all-football-players-stats-in-top-5-leagues-2324) |
| `top5-players-2425.xlsx` | 2024/25 | [Kaggle — Top 5 Leagues 24/25](https://www.kaggle.com/datasets/orkunaktas/all-football-players-stats-in-top-5-leagues-2425) |

## Instrucciones

1. Accede a los enlaces de Kaggle indicados.
2. Descarga el archivo Excel de cada dataset.
3. Renómbralo según la tabla anterior y colócalo en esta carpeta (`data/`).
4. Configura las rutas en el fichero `.env` si decides usar nombres o rutas distintos.
